## ✨ Features

- 🎨 **Modern UI** — Clean, minimal design with smooth animations
- 📱 **Fully Responsive** — Looks great on mobile, tablet & desktop
- ⚡ **Blazing Fast** — Optimized with Next.js SSR & lazy loading
- 🌙 **Dark Mode** — Eye-friendly dark/light theme toggle
- 📬 **Contact Form** — Integrated contact form with backend support
- 🗂️ **Projects Showcase** — Dynamic project cards with live demo & GitHub links
- 🛠️ **Skills Section** — Visual tech stack display
- 📄 **Resume Download** — One-click resume/CV download

## 📦 Installation & Setup

```bash
# Clone the repo
git clone https://github.com/khushalkumarsahu/portfolio.git

# Go into the project directory
cd portfolio

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
npm install
```

### ▶️ Run Locally

```bash
# Start backend server
cd server
npm run dev

# Start frontend (in a new terminal)
cd client
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. 🎉
